Propósito del sitio

Este sitio web ha sido creado con la finalidad de resolver la problemática siguiente:

A raíz de la pandemia del COVID-19, la educación virtual ha sustituido la necesidad de la presencialidad, siendo usado por todas las instituciones educativas, esto ha traído como resultado la necesidad de elaborar recursos educativos digitales para dinamizar las estrategias de aprendizaje. Una estrategia la constituyen los Objetos virtuales de aprendizaje. Estos recursos son desarrollados por empresas que diseñan contenidos multimediales e interactivos, basados en HTML5, CSS3 entre otras tecnologías.

Dentro del presente sitio web, se desarrolla un objeto virtual de aprendizaje (OVA) que explica las temáticas de la aplicación de estilos CSS a una pagina web HTML y se analiza y describe el proceso de interaccion humano ordenador, esto para brindar material de apoyo de calidad a la universidad para la educacion de sus estudiantes.

Temáticas del OVA

Dentro de este sitio web, se presenta material de apoyo para el estudiante, que aportará a su adquisición de habilidades en la creación y diseño de sitios web, aplicando las mejores prácticas en los procedimientos relacionados.

En específico, dentro del sitio, podrá encontrar información referente a los siguientes temas:

  1. Sintaxis y selectores CSS

    Se conocerá la importancia y la estructura de CSS para organizar y aplicar estilos visuales que ayuden a mejorar la visualización de una pagina web a los usuarios.
    Se estudia la estructura básica de CSS, así como las tres principales formas de agregar estilos CSS a un documento HTML.

  2. Interacción Humano-Ordenador

    Se analiza la definición e importancia de la interacción Humano Ordenador (HCI), así como el impacto que esta tiene en el desarrollo de sistemas informaticos e interfaces de interacción del humano con las computadoras y demás dispositivos tecnológicos.
    Se conocerán los aspectos estudiados por la HCI, así como los distintos paradigmas que se tienen en cuenta para su diseño.

También podrá contar con recursos multimedia que refuercen el aprendizaje de las temáticas a trabajar, así como con actividades evaluativas que le permitan al estudiante practicar lo aprendido.
