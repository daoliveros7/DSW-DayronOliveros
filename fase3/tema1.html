    Fundamentos de CSS

    ¿Qué es CSS?

    CSS (Cascade Style Sheets) corresponde a un lenguaje de programación que es bastante similar a HTML que hace posible que se puedan aplicar estilos a los distintos elementos que componen una pagina web, de forma tal que los títulos, listas y párrafos puedan verse igual en todas y cada una de las páginas. (Celaya, 2014). De esta forma, se garantiza una uniformidad de estilos visuales en los sitios web que se desarrollen.


    Sintaxis básica

    Los estilos CSS se componen de tres partes: un selector, una propiedad y un valor, y su sintaxis es la siguiente:

    {propiedad:valor;}

    • Selector: Nos permite indicar a qué etiqueta HTML se aplicará un estilo.
    • Propiedad: Nos permite indicar qué propiedad de la etiqueta va a aplicarse al estilo, por ejemplo, color de fondo, fuente, tamaño y estilo de letra, orientación, entre otros.
    • Valor: Define el valor de la propiedad relacionada.

    Estructura básica de CSS.

    Ejemplo:

    body {background-color: red;}
    En este estilo, se define que el color de fondo de la página será rojo. El selector corresponde a body, la propiedad en este caso es background-color: y el valor de la propiedad es red.
    Es posible definir más de una propiedad dentro de un mismo estilo CSS, separando cada una usando punto y coma, por ejemplo:

    h1{text-align:justify; color:green;}
    En este caso, se define la alineación justificada del texto y el color verde para todos los títulos de nivel 1.

    Agrupación de elementos y estilos

    De acuerdo con las palabras de Celaya, es posible definir estilos diferentes para un mismo tipo de etiqueta HTML, creando clases mediante el atributo class y luego indicando en CSS las propiedades de cada clase en particular.
    Por ejemplo, en una página web hay dos listas de enlaces, una de comidas y otra de bebidas: 

                        <p> Comidas: </p>
                        <ul>
                            <li> <a href=”sopas.htm”>Sopas</a></li>
                            <li> <a href=”entradas.htm”>Entradas</a></li>
                            <li> <a href=”fuertes.htm”>Platos fuertes</a></li>
                        </ul>
                    <p> Bebidas: </p>
                        <ul>
                            <li> <a href=”jugos.htm”>Jugos</a></li>
                            <li> <a href=”cervezas.htm”>Cervezas</a></li>
                            <li> <a href=”calientes.htm”>Bebidas calientes</a></li>
                    </ul>

    Supongamos que se necesita que los enlaces de las comidas aparezcan en color verde, mientras que los de las bebidas en color naranja. El resto de los enlaces en esta página deberán aparecer en color azul. Con el atrubito class, se definen dos clases: comidas y bebidas, como propiedades dentro de la etiqueta <a> que define los enlaces: 

                        <p> Comidas: </p>
                        <ul>
                            <li> <a href=”sopas.htm” class=”comidas”>Sopas</a></li>
                            <li> <a href=”entradas.htm” class=”comidas”>Entradas</a></li>
                            <li> <a href=”fuertes.htm” class=”comidas”>Platos fuertes</a></li>
                        </ul>
                    <p> Bebidas: </p>
                        <ul>
                            <li> <a href=”jugos.htm” class=”bebidas”>Jugos</a></li>
                            <li> <a href=”cervezas.htm” class=”bebidas”>Cervezas</a></li>
                            <li> <a href=”calientes.htm” class=”bebidas”>Bebidas calientes</a></li>
                        </ul>

    Se utiliza la siguiente sintaxis CSS para asignar las propiedades de cada clase:

    a{color:blue;}
    a.comidas{color:green;}
    a.bebidas{color:orange;}

    Identificación de estilos con el atributo ID

    Dentro de cada etiqueta HTML, es posible incluir un identificador por medio del atributo ID. No pueden existir dos elementos dentro del mismo documento con el mismo id. Este id sirve para aplicar un estilo especial solamente a un elemento de la página. En CSS, se indica que un estilo se asigna a un identificador concreto usando el carácter numeral #. (Celaya, 2014).
    Por ejemplo, en la siguiente disposición de líneas de texto de tipo h1 y h2, se tienen los nombres de los capítulos de un libro. Como se observa, cada línea lleva un id diferente.

                    <h1 id=”c1”> Capítulo 1 </h1>
                    <h2 id=”c1-1”> Capitulo 1.1 </h2>
                    <h2 id=”c1-2”> Capitulo 1.2 </h2>
                <h1 id=”c2”> Capítulo 2 </h1>
                    <h2 id=”c2-1”> Capitulo 2.1 </h2>
                    <h2 id=”c2-2”> Capitulo 2.2 </h2>
                …

    Si quisiéramos aplicar un formato especial, por ejemplo, color azul solamente al capítulo 2-1, se escribiría la sintaxis CSS de la siguiente forma:
    #c2-1 {color: blue;}

    ¿Cómo se aplica CSS a un documento HTML?

    Existen tres formas de aplicar estilos CSS dentro de HTML, siendo la opción número 3 la que más se recomienda aplicar:

    1. Usando el atributo Style

    El atributo style es de CSS, pero se puede insertar dentro de etiquetas HTML directamente. Por ejemplo, para aplicar el color verde al fondo de la página, en el documento HTML se escribiría, dentro de la etiqueta body lo siguiente: 

                            <html>
                        <head>
                                <title>Título de mi Web</title>
                        </head>
                        <body style=”background-color:green;”>
                            <p>Página web con fondo verde.</p>
                        </body>
                        </html>


    2. Por medio de la etiqueta style

    En esta forma, se inserta también código CSS dentro de un archivo HTML, mediante una etiqueta HTML llamada style. Por ejemplo, para conseguir el mismo resultado que en el caso 1, se escribe: 
                            <html>
                        <head>
                                <title>Título de mi Web</title>
                                
                                <style type=”text/css”>
                                    body {background-color:green;}
                                </style>
                                
                        </head>
                        <body>
                            <p>Página web con fondo verde.</p>
                        </body>
                        </html>
    3. Por medio de archivo CSS externo

    En esta forma, se puede ahorrar mucho trabajo, por eso, esta es la forma más recomendable para aplicar estilos CSS. Por ejemplo, Si se desea cambiar el color de fondo de un sitio web compuesto por 100 páginas, se puede llevar a cabo en unos segundos modificando parte del código de la hoja de estilo principal en lugar de hacerlo en cada página individual.
    Una hoja de estilos CSS consiste en un archivo de texto, al igual que un archivo HTML, solo que presenta la extensión .css. Por tanto, la forma de crearlas y editarlas es igual que con los archivos HTML, y se puede usar también el bloc de notas o cualquier editor de texto. En el archivo estarán escritas todas las reglas de estilos en lenguaje CSS, guardando el archivo con un nombre como por ejemplo estilos.css.
    Después, en cada archivo HTML al que se le quiera aplicar la hoja de estilos, simplemente es necesario insertar una línea de código dentro del encabezado, indicando la ruta del archivo CSS, de la siguiente forma: 
                            <html>
                        <head>
                                <title>Título de mi Web</title>
                                
                                <link rel=”stylesheet” type=”text/css” hfref=”estilos.css” />
                                
                        </head>
                        <body>
                        …
